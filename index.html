NOVAXML.JS

Trött på fula schemaappar? Den här sidan erbjuder ett skript som analyserar ditt schema och genererar en fil din telefon kan läsa av.

FAQ:

Min skola finns inte! Vad ska jag göra? Jag måste ha mitt schema. NU.

Det är klart alla skolor måste ha tillgång till sitt schema överallt och alltid! Om du inte vet hur GitHub fungerar kan du skicka oss ett mail med en länk till schemat.

Om du vill utmana dig lite kan du kolla instruktionerna på vår GitHub-sida och sedan bara göra en PR till vårt projekt.

Varför frågar ni om personnummer? Jag vill helst inte dela med mig av sånt.

För att läsa av schemat behöver vi scheman som inte innehåller överlappande lektioner. Ett exempel på en överlappande lektioner kan vara individuella val, där en hel klass kan ha olika lektioner samtidigt. För att förhindra detta genererar vi ett personligt schema. Om du inte vill skriva in ditt personnummer så behöver du inte, men räkna inte med att få ett xml-schema då.

Sparar ni personnummrerna någonstans, säljer de vidare eller gör något annat ondskefullt?

Det enda sättet att verkligen bevisa att att vi inte gör detta är att ta sig en titt i källkoden. Hela den här sidan är publicerad på github (kolla längst ner på sidan för en länk). Om du inte kan läsa kod så får du helt enkelt lite på oss när vi säger att vi inte sparar/säljer/gör ondskefulla saker med det du skriver in.

Vi tycker att vår kod är så bra och godhjärtad att vi vill skryta om den för hela världen. Därför finns allt publicerat på github.
